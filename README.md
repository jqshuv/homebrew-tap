## How do I install these formulae?

`brew install jqshuv/tap/<formula>`

Or `brew tap jqshuv/tap` and then `brew install <formula>`.

Or, in a [`brew bundle`](https://github.com/Homebrew/homebrew-bundle) `Brewfile`:

```ruby
tap "jqshuv/tap"
brew "<formula>"
```
